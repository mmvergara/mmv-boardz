## Installation

### Client

- Change Config at ./src/Config.tsx
- npm install & npm start

### Server

- Fill out .env file

```jsx
MONGODB_URI=
SECRET_KEY_EXP=
HOST=
PORT=
```

- npm install
- npm run dev / npm start
